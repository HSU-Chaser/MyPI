Returns the number of series in the chart object.

Visual Basic
Public ReadOnly Property SeriesCount As Integer
C#
public int SeriesCount { get; }

Property Value

An integer value containing the series count.

Code Sample

The following code illustrates use of the SeriesCount property.

Visual Basic
' Add 2 consecutive series
chart.SetSeriesValuesFromString( 0, "34;56;23;76;12" )
chart.SetSeriesValuesFromString( 1, "76;56;34;12;23" )

Dim count As Integer = chart.SeriesCount
' count is equal to 2
Me.Response.Write( "Number of series in chart= " + count + "<br>" )

' Add another series with series index 4
' (series 2 and 3 remain empty and not displayed)
chart.SetSeriesValuesFromString( 4, "56;34;12;23;76" )

count = chart.SeriesCount
' count is now equal to 3
Me.Response.Write( "Number of series in chart= " + count + "<br>" )
