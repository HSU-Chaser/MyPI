Specifies the title of a chart (displayed on top of a chart).

void object->SetTitle( string title )

Arguments

object
Required. A Chart object.
title
Required. A string that specifies the title.

Remarks

By default, a new chart has no title.

PHP Sample Code

The following PHP code illustrates use of the SetTitle method. 

// Create the chart series
$values_1= array();
$values_2= array();

$values_1[0] = 10;
$values_1[1] = 20;
$values_1[2] = 30;

$values_2[0] = 100;
$values_2[1] = 200;
$values_2[2] = 300;

$chart->SetSeriesValuesFromArray(0, $values_1);
$chart->SetSeriesValuesFromArray(1, $values_2);

// Set the chart title
$chart->SetTitle("My Chart");

// Set the graphics settings from a style file
$chart->LoadStyle("C:\\myStyle.scs");

// The chart is complete, we can now generate the movie
$chart->ExportAsResponse();
