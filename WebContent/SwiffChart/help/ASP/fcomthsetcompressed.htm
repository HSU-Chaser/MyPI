Specifies whether the generated content (Flash or SVG) should be compressed or not. 

object.SetCompressed compressed

Arguments

object
Required. A Chart object.
compressed
Required. A boolean value that specifies whether the generated content (Flash or SVG) is compressed or not.

Remarks

This option only applies to Flash or SVG generated content.

Default: the generated content is not compressed.

Note: Flash Player 6 or higher is required for playing back a compressed Flash movie.

VBScript Sample Code

The following VBScript code illustrates use of the SetCompressed method in an ASP page. 

chart.SetDataFromQuery
chart.LoadStyle "C:\myStyle.scs"

'Let's display a compressed SWF movie 
chart.SetCompressed true
chart.ExportAsResponse
