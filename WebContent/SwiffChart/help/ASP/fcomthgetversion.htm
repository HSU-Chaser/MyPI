Retrieves the version of a chart object.

object.GetVersion

Arguments

object
Required. A Chart object.

Return Value

The GetVersion method returns a string value that contains the version of the Swiff Chart Generator.
The string is formatted as "Major.Minor.Level.SubLevel". Example: "3.0.0.1".

VBScript Sample Code

The following VBScript code illustrates use of the GetVersion method in an ASP page. 

'Create a new chart object 
Dim chart
Set chart= Server.CreateObject("SwiffChartObject.ChartObj")

'Display its version in the HTML page 
Response.Write chart.GetVersion
Response.Write "<br>"
