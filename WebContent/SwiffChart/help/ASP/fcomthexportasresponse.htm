Generates a Flash movie or an image from a chart object and writes it to the Response intrinsic object.

object.ExportAsResponse

Arguments

object
Required. A Chart object.

Remarks

This method should be used within an ASP environment with the Response object provided by the web server. The method actually:

  • generates the content (SWF, PNG, JPG, SVG or PDF data) from the chart object
  • clears the Response object, i.e. Response.Clear
  • defines the content type, i.e. Response.ContentType = "application/x-shockwave-flash" for Flash, "image/jpeg" for JPEG image, "image/png" for PNG image, "image/svg-xml" for SVG document, or "application/pdf" for PDF document
  • writes the binary data (movie, image or document) to the Response object

The format of the generated content can be specified by the method SetOutputFormat.

VBScript Sample Code

The following VBScript code illustrates use of the ExportAsResponse method. 

'Create the chart series
Dim values_1(2)
Dim values_2(2)

values_1(0) = 10
values_1(1) = 20
values_1(2) = 30

values_2(0) = 100
values_2(1) = 200
values_2(2) = 300

chart.SetSeriesValuesFromArray 0, values_1
chart.SetSeriesValuesFromArray 1, values_2

'Set the graphics settings from a style file
chart.LoadStyle "C:\myStyle.scs"

'The chart is complete, we can now write the Flash movie into the response object
chart.ExportAsResponse
