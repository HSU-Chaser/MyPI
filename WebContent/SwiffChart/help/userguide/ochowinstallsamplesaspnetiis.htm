Introduction

To make the Swiff Chart Generator ASP.NET 2.0 samples available on your Microsoft Internet Information Server (IIS) you need to create a new Website pointing to the disk location of the Swiff Chart Generator ASP.NET 2.0 sample website, usually located in C:\Program Files\GlobFX\Swiff Chart Generator 3\samples\ASP.NET 2.0\WebSite.


Samples Installation

To create a new website pointing to the Swiff Chart Generator sample website on Microsoft Internet Information Server (IIS), please follow these simple steps:

  1. Click Start, point to Administrative Tools, and then click Internet Information Services (IIS) Manager.
  2. In Internet Information Services (IIS) Manager, expand the server name, right-click Sites, and then click Add Web Site.
  3. In the Add Web Site dialog box, add the information for the new Web site. When you do, ensure that you do the following:
    - In the Physical path box, specify the path where the samples are installed, usually C:\Program Files\GlobFX\Swiff Chart Generator 3\samples\ASP.NET 2.0\WebSite.
    - To not conflict with the Default Web Site, you should either specify a different IP address in the IP address list, or specify a port other than port 80 in the Port box.
  4. When you are finished, click OK.
  5. Under the Sites node, right-click the newly created website and select Browse to launch the Swiff Chart Generator sample website.

Now, you can use the samples alias to get to the files stored in the Swiff Chart Generator samples directory.


Testing Swiff Chart Generator

To test the Swiff Chart Generator installation, open your web browser (you must have the Flash plug-in installed) and type the following URL in:

http://localhost:<port_number>/

where port_number is the port number of the newly created website pointing to the Swiff Chart Generator samples.

An HTML page should appear, providing you with the list of samples.
