After a successful installation of Swiff Chart Generator you should create the samples virtual directory on your web server - this is described on the Installation of the Samples (install.htm) section. You can then visit the included Sample to understand the possibilities offered by Swiff Chart Generator.

Windows: Swiff Chart Generator for Windows is implemented as a COM Object that is automatically registered when Swiff Chart Generator is installed. Furthermore, an ASP.NET module (the Swiff Chart Web Control) will also be installed.

PHP Users Note: Your PHP scripts will require SwiffChart.php. Simply manually install this file near your own PHP script or in your system configuration. Your will find this file in the samples directory.

JSP Users Note: Your JSP scripts will require SwiffChart.jar. Simply manually install this JAR archive on your Web Application Server and make it accessible from your JSP scripts for use page import directive.


Tip:

To install Swiff Chart Generator without any user interaction (automatic or remote), simply issue the following command: 

C:\> SwiffChartGenSetup.exe /SP- /VERYSILENT
