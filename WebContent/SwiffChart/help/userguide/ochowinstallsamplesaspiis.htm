Introduction

To make the Swiff Chart Generator Active Server Pages (ASP) samples available on your Microsoft Internet Information Server (IIS) you need to create a Virtual Directory (also known as web-sharing a directory) pointing to the disk location of the Swiff Chart Generator samples, usually C:\Program Files\GlobFX\Swiff Chart Generator 3\samples\.


Samples Installation

To web-share the Swiff Chart Generator samples on Microsoft Internet Information Server (IIS), please follow these simple steps:

  1. Click the Start button, point to Programs, point to Administrative Tools, and click Computer Management.
  2. Under the Server Applications and Services node, expand Internet Information Services.
  3. In the Internet Information Services snap-in, select the Web site to which you want to add the Swiff Chart Generator samples directory.
  4. Click the Action button, and then point to New, and select Virtual Directory.
  5. Use the New Virtual Directory Wizard to complete this task.
    - In the Directory box, enter the your Swiff Chart Generator samples directory, usually C:\Program Files\GlobFX\Swiff Chart Generator 3\samples\
    - In the Alias box, enter samples

Now, you can use the samples alias to get to the files stored in the Swiff Chart Generator samples directory.


Testing Swiff Chart Generator

To test the Swiff Chart Generator installation, open your web browser (you must have the Flash plug-in installed) and type the following URL in:

http://your_server_name/samples/index.html

where yourserver is the machine name of your web server (possibly 127.0.0.1 - the localhost machine)

An HTML page should appear, providing you with the list of samples.

Note: IIS uses the alias samples to refer to the physical directory [drive letter]:\Program Files\GlobFX\Swiff Chart Generator 3\samples\ When you run an ASP page, you must type its URL in the browser's address window. You can't load the file into the browser because it won't execute the ASP features unless you access it via IIS.
